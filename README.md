<br>

# 仕様

## メニュー
メインメニューは<br>
クエスト / バイト / ショップ / 設定<br>

## クエスト
入力で選択肢を選びダンジョンを進む<br>
選択肢は<br>
進む / 休む / アイテム / マップ / 撤退<br>

## 戦闘
パラメータは LV HP MP ATK MAT DEF MDF SPD LUC<br>
コマンドは 攻撃 魔法 アイテム 逃げる<br>
属性は 無属性 水 炎 風 土 闇 光<br>
特殊効果は いっぱい<br>
敵は 1～∞<br>
味方は 1人(データの仕様)
